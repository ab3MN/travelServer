https://brainy-tutu.cyclic.app

    User model:

  • password: String required
  • email: String required
  • avatar: String
    default:img size 80px
/users/signup

    await email password

  • if no email return status 401 Email is required
  • if no password return status 401 Password is required
  • if email in use return status 409 Email in use
  • if OK return status 201 + User + accessToken + refreshToken
/users/login

    await email password

  • if no email return status 401 Email is required
  • if no password return status 401 Password is required
  • if wrong email return status 401 Email is wrong
  • if wrong password return status 401 Password is wrong
  • if OK return status 200 + User + accessToken + refreshToken
/users/auth

    await refresh token

  • if no token return status 401 Not authorized
  • if wrong token return status 403 Bad token
  • if Ok return status 200 + User
/users/logout

    await refresh token

  • if no token return status 401 Not authorized
  • if wrong token return status 403 Bad token
  • if OK return status 204
